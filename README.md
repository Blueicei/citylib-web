# citylib-web2

## Project setup
```
yarn install
or （如果报错提示node等版本过高，执行下面）
npm install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
